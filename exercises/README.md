# 課題の提出先

## フォルダーを作る

- docs 内のファイル名と同じ名前のフォルダーを作ります。
  - 02_basics.md の課題なら 02_basics というフォルダー名
  - 内部の課題ごとに 01 02 というようにフォルダーを作り、その中で新しいプロジェクトを作ってください

### 禁止事項

- 半角の英数記号以外を用いてはいけません

## ファイルの提出

- フォルダー内に移動して dotnet run で実行できる状態としてください。
- ビルド時などに生成されるファイルがコミットされないように `.gitignore` ファイルを適切に維持してください
